# 🌟 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a simple yet powerful web app designed to help you stay connected with your friends. It reminds you when to reach out, track interactions, and maintain meaningful relationships effortlessly.

---

## 🚀 Description

In today’s busy life, it's easy to lose touch with friends. **KeenKeeper** helps you manage and nurture your friendships by tracking your communication habits and reminding you when it's time to reconnect.

---

## 🛠️ Technologies Used

- ⚛️ Next.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🧠 React Context API

---

## ✨ Key Features

### 🔔 Smart Interaction Tracking

Track your calls, texts, and video chats with friends in one place.

### 📅 Reminder System

Get notified when it's time to reconnect with a friend based on your set goals.

### 📊 Visual Timeline & Insights

View your interaction history with filters and charts for better understanding.
